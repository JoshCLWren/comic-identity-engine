### URL Structure Analysis

**UNIQUE ENCODING**: Uses `1-1` to represent issue `-1`!

```
/price-guide/us/marvel/comic/x-men-1991/1-1/?keywords
```

The `1-1` appears to be:
- First `1`: Series identifier or position
- Second `1`: Issue number (-1 encoded as 1-1)
- This is the most unusual encoding seen across all platforms
